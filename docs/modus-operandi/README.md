- **`CLAUDE.md`** (raíz) = reglas operativas vigentes, la verdad actual. Es lo primero que se lee.

## Regla de mantención

- Si una **convención de estilo** cambia o mejora → actualizar `convenciones-tarroshorts.md` en el mismo commit del cambio. No dejar la mejora solo en el último HTML.
- Si se **toma una decisión** con Luis (formato, estrategia, regla nueva) → agregar una entrada al final de `bitacora-decisiones.md`.
- Fuente canónica = el repo. El respaldo al Drive (`G:\Mi unidad\Studio\`) lo deja el sync — no editar la copia del Drive a mano.
