# Retrotarros — Identidad visual

## Dirección estética

**Synthwave / arcade retro.**

## Paleta de marca (usar literal estos hex)

| Rol | Color | Hex |
|-----|-------|-----|
| Primario, acentos | Magenta neón | `#FF2E88` |
| Secundario, contraste | Cyan eléctrico | `#00E5FF` |
| Fondo principal | Morado profundo | `#2D1B69` |
| Etiquetas, Indie Lat, CTAs | Amarillo arcade | `#FFD23F` |
| Texto sobre oscuro | Blanco hueso | `#F5F0E8` |

No usar `#FFFFFF` puro — siempre el blanco hueso `#F5F0E8`.

## Tipografía

| Uso | Fuente |
|-----|--------|
| Logo | Monoton, Press Start 2P (tratamiento neón, pixel art retro-futurista) |
| Títulos de miniatura | Bebas Neue o Anton (condensadas, pesadas) |
| Texto general | Inter o Montserrat |

## Miniaturas

- Rostro de Luis o Koko en **primer plano**.
- Un elemento del juego identificable.
- Título en **2–4 palabras** con tipografía pesada y contorno.
- Etiqueta amarillo arcade `#FFD23F` en esquina superior con texto "INDIE LAT" cuando aplique.
